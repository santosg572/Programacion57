NAME
       zmq_msg_data - retrieve pointer to message content

SYNOPSIS
       void *zmq_msg_data (zmq_msg_t *msg);

DESCRIPTION
       The zmq_msg_data() function shall return a pointer to the message content of the message object referenced
       by msg.

           Caution
           Never access zmq_msg_t members directly, instead always use the zmq_msg family of functions.

RETURN VALUE
       Upon successful completion, zmq_msg_data() shall return a pointer to the message content.

ERRORS
       No errors are defined.
